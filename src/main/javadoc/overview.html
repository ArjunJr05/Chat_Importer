The Arattai WhatsApp Chat Parser provides a robust pipeline for converting raw WhatsApp backups into structured Java objects. It handles:

  • Regex-based Transcript Parsing: Accurately identifies senders, timestamps, and multi-line message bodies.
  • Media Discovery: Automatically scans the ZIP archive to link attachments (Images, Videos, Voice Notes) to their corresponding messages.
  • Binary Metadata Extraction: Deep-dives into files to extract image resolutions and media durations without external dependencies.
